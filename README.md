# GroupTab

Key Features of your project to highlight:
Settlement Logic: Includes a custom algorithm to calculate "who owes who" across multiple receipts.

Tech Stack: Utilizes a modern stack including React (Vite) for the frontend, Flask for the backend API, and Firebase for authentication and database management.

Collaboration: Supports shared "trips" where multiple users can join and contribute via unique invite codes.

Customization: Allows users to personalize trip covers and receipt themes with custom images and gradients.
